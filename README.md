# React Pizza
React pizza - інтернет магазин піцц. Зроблено на основі курсу автора [Archakov06](https://github.com/Archakov06).

## Стек
- ReactJS
- Redux
- Redux-thunk
- React Router
- Axios

## Запуск
```git clone https://github.com/VladSvezhentsev/react-pizza.git```

```npm install```

```npm start```

```npm run dev```
